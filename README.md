# FILE-COMMANDER

## File Metadata

Every file on your computer has metadata — information about the file itself. Here is what you can find:

1. **File name**
2. **File size** — how many bytes
3. **File type** — .py, .md, .jpg, etc.
4. **Date created**
5. **Date last modified**
6. **Date last accessed**
7. **Owner** — who the file belongs to on the computer
8. **Permissions** — who can read, write, or run it
9. **Location** — the full path on your computer

For image files (.jpg, .png) there is extra metadata called EXIF:

10. **Camera model**
11. **Date photo was taken**
12. **GPS location** — where the photo was taken
13. **Image size** — width and height in pixels
